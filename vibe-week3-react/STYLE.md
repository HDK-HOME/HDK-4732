# STYLE.md

## Design Style

- Clean
- Simple
- Modern
- Beginner-readable
- Not too flashy

## UI Direction

Use clear spacing, readable font sizes, and simple card-style sections.

Avoid making the first screen feel like a marketing landing page unless the user asks for one. This is a practice app first.

## Copy Style

- Korean-friendly
- Direct
- Practical
- Mentor-like
- Avoid AI-sounding text
- Explain what the user can see and change

## Code Style

- Use simple React function components.
- Use clear variable names.
- Keep comments minimal and useful.
- Avoid clever code.
- Keep `src/App.tsx` easy to scan.
- Keep `src/App.css` organized by visible sections.
- Use TypeScript types only when they make the code easier to understand.

## Component Style

Do not split components too early.

Start in `src/App.tsx`. Create a new component only when it makes the lesson clearer.

When props are introduced, use simple prop types or simple interfaces.
