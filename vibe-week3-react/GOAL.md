# GOAL.md

## Project Goal

Build a beginner-friendly React + Vite + TypeScript practice project based on Vibe Coding Week 3.

## Learning Goals

- Understand React components.
- Understand JSX.
- Understand how `App.tsx` controls the screen.
- Understand how CSS affects React UI.
- Practice changing UI step by step.
- Learn simple TypeScript usage in React.
- Learn how to run, test, commit, and push changes.

## First Milestone

Create a simple React page with:

- title
- description
- button
- card layout
- basic CSS styling

## Later Milestones

- Add state with `useState`.
- Add input fields.
- Add button click interactions.
- Add simple list rendering.
- Add component separation only when it helps learning.
- Add simple prop types after component separation.

## Success Criteria

- A beginner can understand the main files.
- The project stays simple.
- `src/App.tsx` and `src/App.css` remain the default practice files.
- TypeScript types stay simple and helpful.
- `npm run build` succeeds.
- Existing sibling projects stay untouched.
