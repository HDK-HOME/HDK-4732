Privacy Policy

개인정보처리방침

마스터홍 판매신호 크롬 확장 프로그램 및 마스터홍 웹앱의 개인정보 처리 기준입니다.

수집하는 정보

사용자가 직접 연 네이버 상품 페이지에서 공개 상품 정보와 판매신호 후보만 읽습니다. 예를 들어 상품명, 상품 URL, productId, 리뷰 수, 관심고객수, 가격 정보 후보를 분석합니다.

수집하지 않는 정보

로그인 정보, 결제 정보, 쿠키, 비밀번호, 개인 계정 정보는 수집하지 않습니다. 확장 프로그램은 사용자가 버튼을 누른 현재 탭의 공개 페이지 내용만 분석합니다.

데이터 전달

분석 결과는 사용자가 확장 프로그램에서 버튼을 누를 때만 마스터홍 웹앱으로 전달됩니다. 자동 백그라운드 수집이나 자동 크롤링은 수행하지 않습니다.

저장 방식

서버 저장 기능이 도입되기 전까지 분석 기록은 사용자의 브라우저 localStorage에 저장됩니다. 같은 브라우저 안에서만 기록을 확인할 수 있습니다.
