Terms

이용약관

마스터홍 웹앱과 크롬 확장 프로그램의 기본 이용 조건입니다.

서비스 목적

마스터홍은 네이버 스마트스토어 셀러가 검색 순위와 공개 상품 신호를 참고해 개선 우선순위를 판단하도록 돕는 분석 도구입니다.

분석 결과의 성격

순위와 판매신호는 조회 시점, 검색 API 응답, 공개 상품 페이지에서 확인 가능한 정보에 따라 달라질 수 있습니다. 판매량은 공식 확정 데이터가 아닌 참고 지표입니다.

사용자 책임

사용자는 분석 결과를 사업 판단의 보조 자료로 활용해야 하며, 가격 변경, 광고 집행, 상품 정보 수정 등 최종 의사결정 책임은 사용자에게 있습니다.

API와 확장 프로그램 이용

순위 확인 기능은 입력한 키워드, 스토어명, 조회 범위를 바탕으로 동작합니다. 크롬 확장 프로그램은 사용자가 직접 연 상품 페이지에서 버튼을 눌렀을 때만 공개 페이지 정보를 웹앱으로 전달합니다.

제한 사항

서비스를 과도한 자동 요청, 비정상 접근, 타인의 권리 침해, 관련 법령 위반 목적으로 사용할 수 없습니다.

서비스 상태

마스터홍은 출시 준비 중인 무료 도구이며, 기능과 화면은 베타 운영 과정에서 바뀔 수 있습니다. 중요한 변경은 웹사이트 문서에 반영합니다.

변경

서비스 기능과 약관은 운영 상황에 따라 변경될 수 있으며, 중요한 변경은 웹사이트에 게시합니다.
