# 마스터홍 판매신호 웹스토어 제출 준비

## 기본 정보

- 서비스명: 마스터홍
- 도구명: 판매신호 분석기
- 확장 프로그램명: 마스터홍 판매신호
- 영문 표기: Master Hong
- 카테고리 후보: 쇼핑 또는 생산성

## 짧은 설명

네이버 상품 페이지에서 공개 판매신호, 리뷰 수, 관심고객수, 가격 정보를 감지해 마스터홍 판매신호 분석기에서 분석합니다.

## 상세 설명 초안

마스터홍 판매신호는 네이버 스마트스토어, 브랜드스토어, 쇼핑 상품 페이지에서 사용자가 직접 열어본 공개 정보를 기반으로 판매신호를 분석합니다.

확장 프로그램에서 `이 상품 분석하기`를 누르면 상품명, 스토어명, 판매신호, 리뷰 수, 관심고객수, 가격 정보를 감지하고 마스터홍 판매신호 분석기에서 결과를 보여줍니다.

마스터홍은 네이버 공식 판매량 지표를 제공한다고 주장하지 않습니다. 분석 결과는 공개 페이지에서 감지한 판매신호와 추정/공개 신호입니다.

## 권한 사용 사유

- `activeTab`: 사용자가 버튼을 누른 현재 네이버 상품 페이지를 분석하기 위해 필요합니다.
- `scripting`: 현재 탭에 content script를 주입해 공개 페이지 텍스트와 script 데이터를 읽기 위해 필요합니다.
- `host_permissions`: 네이버 스마트스토어, 브랜드스토어, 쇼핑 페이지에서만 동작하도록 제한합니다.

## 개인정보 처리 요약

- 사용자가 직접 연 네이버 상품 페이지의 공개 상품 정보와 판매신호 후보만 읽습니다.
- 로그인 정보, 결제 정보, 쿠키, 비밀번호는 수집하지 않습니다.
- 분석 결과는 사용자가 버튼을 누를 때만 마스터홍 웹앱으로 전달됩니다.
- 서버 저장 전까지 기록은 브라우저 localStorage 기반입니다.

## 제출 전 준비물

- 128x128 확장 아이콘
- 스토어 스크린샷
- 홍보 이미지
- 실제 개인정보처리방침 URL
- 실제 문의 이메일
- 확장 zip: `dist/master-hong-extension.zip`
- 테스트용 네이버 상품 페이지 URL

## 최종 확인

- 확장 이름이 `마스터홍 판매신호`인지 확인
- 설명에서 기존 브랜드 표현이 없는지 확인
- 네이버 공식 판매량이 아니라 공개 판매신호라는 표현 유지
- `npm run package:extension`으로 zip 생성 확인
