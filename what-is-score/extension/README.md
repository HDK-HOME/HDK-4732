# 마스터홍 판매신호 Chrome Extension

마스터홍 판매신호는 네이버 상품 페이지에서 공개 판매신호, 리뷰 수, 관심고객수, 가격 정보를 감지해 마스터홍 판매신호 분석기에서 분석하는 소비자용 MVP입니다.

영문 표기: Master Hong

## 개발자 테스트 설치

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 오른쪽 위 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 누릅니다.
4. `what-is-score/extension` 폴더를 선택합니다.
5. 네이버 상품 페이지를 엽니다.
6. 확장 팝업에서 `이 상품 분석하기`를 누릅니다.

## 사용자 흐름

1. 사용자가 네이버 상품 페이지를 엽니다.
2. 마스터홍 판매신호에서 `이 상품 분석하기`를 누릅니다.
3. 확장이 현재 페이지의 공개 신호를 감지합니다.
4. 마스터홍 웹앱이 새 탭으로 열립니다.
5. 판매신호 분석기가 결과 카드와 기록을 자동으로 표시합니다.

## 수집 및 전송 데이터

전송 payload는 소비자용으로 제한합니다.

- `productId`
- `currentUrl`
- `collectedAt`
- `normalizedSignals`

`detectedSignals`, `bodyTextPreview`, `scriptJsonCandidates`는 웹앱 전송 payload에 포함하지 않습니다.

## Chrome 웹스토어 제출 전 체크리스트

- 확장 이름: `마스터홍 판매신호`
- 서비스명: `마스터홍`
- 도구명: `판매신호 분석기`
- 영문 표기: `Master Hong`
- 아이콘 최종 확정
- `privacy.html`의 문의 이메일을 실제 주소로 교체
- Chrome 웹스토어 상세 설명 작성
- 개인정보 처리방침 URL 준비
- 스크린샷 1,280x800 또는 640x400 준비
- 홍보 이미지, 작은 타일, 큰 타일 이미지 준비
- 권한 사유 설명 작성: `activeTab`, `scripting`, 네이버 쇼핑 도메인 접근
- 테스트 계정 또는 테스트 절차 문서화
- 네이버 공식 판매량이 아닌 공개 페이지 감지 신호라는 표현 확인

## zip 패키징

앱 루트에서 실행합니다.

```bash
cd what-is-score
npm run package:extension
```

결과물:

```text
dist/master-hong-extension.zip
```

- 스크린샷
- 권한 사용 목적 설명

## 제한

- 사용자가 버튼을 누른 현재 탭만 분석합니다.
- 로그인 정보, 쿠키, 결제 정보, 비밀번호는 수집하지 않습니다.
- 감지 결과는 네이버 공식 판매량 지표가 아니라 공개 페이지에서 읽은 추정/공개 신호입니다.
