# 마스터홍 판매신호 베타 설치 안내

## 설치 순서

1. 전달받은 확장 ZIP 파일을 압축 해제합니다.
2. Chrome 주소창에 `chrome://extensions`를 입력합니다.
3. 우측 상단 `개발자 모드`를 켭니다.
4. `압축해제된 확장 프로그램 로드`를 클릭합니다.
5. 압축 해제한 `extension` 폴더를 선택합니다.
6. 네이버 스마트스토어 상품 페이지에 접속합니다.
7. Chrome 우측 상단 확장 아이콘을 클릭합니다.
8. `마스터홍 판매신호`를 선택합니다.
9. `이 상품 분석하기`를 클릭합니다.
10. 새 탭으로 열린 마스터홍 판매신호 분석기에서 결과를 확인합니다.

## 확인할 값

- 상품명
- 스토어명
- 판매신호
- 리뷰 수
- 관심고객수
- 현재가
- 정가 후보
- 할인율

## 주의

- 마스터홍 판매신호는 네이버 공식 판매량 지표가 아니라 공개 페이지에서 감지한 판매신호를 분석합니다.
- 사용자가 버튼을 누른 현재 탭만 분석합니다.
- 로그인 정보, 결제 정보, 쿠키, 비밀번호는 수집하지 않습니다.
